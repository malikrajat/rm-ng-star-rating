# Installation

## Step 1: Install the Package

Choose your preferred package manager:

```bash
# npm
npm install rm-ng-star-rating

# yarn
yarn add rm-ng-star-rating

# pnpm
pnpm add rm-ng-star-rating
```

## Step 2: No Additional Dependencies!

Unlike other star rating libraries, `rm-ng-star-rating` has **zero dependencies**. You're ready to go immediately after installation!

## Dependencies Overview

| Dependency | Version | Purpose |
|------------|---------|---------|
| `@angular/core` | ^18.0.0+ | Angular framework |
| `@angular/common` | ^18.0.0+ | Angular common utilities |

**Minimum Requirements:** Angular 18+
